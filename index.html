4. Pseudo-Clases

a. :root

La pseudo-clase :root representa el elemento raíz del documento (normalmente el elemento html).

:root {
  --main-color: #3498db;
  --secondary-color: #2c3e50;
}

5. Pseudo-Elementos

a. ::first-line

Este es un párrafo de ejemplo que demuestra el pseudo-elemento ::first-line. Solo la primera línea de este párrafo tendrá un estilo diferente. El resto del texto se mostrará con el formato normal.

.first-line::first-line {
  font-weight: bold;
  color: #e74c3c;
}
b. ::first-letter

Este es un párrafo de ejemplo que demuestra el pseudo-elemento ::first-letter. La primera letra de este párrafo tendrá un estilo diferente al resto del texto.

.first-letter::first-letter {
  font-size: 2em;
  color: #2c3e50;
  font-weight: bold;
  float: left;
  margin-right: 5px;
  line-height: 1;
}
c. ::before y d. ::after

Contenido principal

.before-after::before {
  content: "Antes: ";
  font-weight: bold;
  color: #2c3e50;
}

.before-after::after {
  content: " :Después";
  font-weight: bold;
  color: #e74c3c;
}
